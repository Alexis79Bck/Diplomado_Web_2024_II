Ejemplo 1

Sintaxis:

HACER MIENTRAS «DO ..... WHILE»

La estructura repetitiva HACER MIENTRAS «DO ...... WHILE» , esta permitirá ejecutar una lista de instrucciones siempre que la condición de control sea verdadera. Cumple la misma función que la estructura «WHILE», solo que la condición de control se evalua al final del bloque de código, garantizando que se ejecute al menos 1 vez las intrucciones. Sintaxis:

do {

  • INSTRUCION 2
  • INSTRUCION 3
  • INSTRUCION 4
  • INSTRUCION 5
} while ($variable_control);

ARREGLOS «ARRAYS»

Los arreglos, también conocido como vectores o arrays, es un tipo de dato estructurado, ya que puede contener una colección de valores. Para acceder a algún valor de esa colección, se utiliza una variable que cumple el rol de indice o posición. Existe 2 formas de declarar una variable del tipo array. Sintaxis:

Primera Forma: $variable = array(valor1, valor2, valor3, ..... valorN);

Segunda Forma: $variable = [valor1, valor2, valor3, ..... valorN];
